<p>jawab: </P>
<p>Menggunakan konstanta seperti keyId dan keyName itu jauh lebih aman dibanding langsung menulis string seperti 'id' atau 'pizzaName'. Karena, semua kunci JSON punya satu sumber yang pasti, jadi kecil kemungkinan terjadi typo yang susah dilacak. Kalau pun salah, IDE bisa langsung kasih tahu saat kompilasi. Selain itu, kode juga lebih gampang dirawat karena kalau suatu saat struktur JSON berubah, cukup ubah nama konstanta di satu tempat saja, dan semua fungsi seperti fromJson dan toJson otomatis ikut menyesuaikan.</p>
